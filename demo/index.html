chromapakz — RGB + lossless signals, one WebM, no WASM

Generate (or load) an RGBD clip, encode it in-browser via WebCodecs VP9, then decode and view it back. The RGB pane is what a legacy player shows; depth (and other uint16 signals) ride along, bit-exact.

This browser doesn't expose VideoEncoder (WebCodecs). Try Chrome, Edge, Firefox 130+, or Safari 26+.
